# Ticket #0003 - Landing Page MVP

## Estado

Completado — implementado en `app/page.tsx`

---

## Objetivo

Reemplazar completamente la página inicial creada por Next.js.

La landing debe comunicar en menos de 5 segundos qué hace MOTivo.

No buscamos un diseño complejo.

Buscamos claridad.

---

## Alcance

Debe incluir únicamente:

- Logo temporal con el texto "MOTivo"
- Título principal:
  "Encuentra la moto perfecta para ti."

- Subtítulo:
  "Responde unas preguntas y recibe una recomendación objetiva basada en tu presupuesto, experiencia y necesidades."

- Botón principal:
  "Empezar"

- Sección "¿Qué hace MOTivo?"

Con cuatro tarjetas:

- Recomendaciones inteligentes
- Comparación objetiva
- Costos reales
- Explicaciones claras

---

## Restricciones

No agregar:

- Login
- Registro
- Navbar
- Footer
- Base de datos
- Backend
- API
- Autenticación
- Librerías nuevas

Usar únicamente:

- Next.js 16
- TypeScript
- Tailwind CSS

---

## Criterios de aceptación

- Responsive
- Código limpio
- Componentes reutilizables
- Fácil de entender
- Sin código innecesario
