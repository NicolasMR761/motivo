# Ticket #0003 - Landing Page MVP

## Estado

En progreso

---

## Objetivo

Reemplazar completamente la página inicial creada por Next.js.

La landing debe comunicar en menos de 5 segundos qué hace MOTivo.

No buscamos un diseño complejo.

Buscamos claridad.

---

## Alcance

Debe incluir únicamente:

- Logo temporal con el texto "MOTivo"
- Título principal:
  "Encuentra la moto perfecta para ti"

- Subtítulo:
  "Recomendaciones inteligentes según tu presupuesto, experiencia y necesidades."

- Botón principal:
  "Empezar"

- Sección "Próximamente"

Con cuatro tarjetas:

- Comparador
- Costos reales
- Mantenimiento
- Consumo

---

## Restricciones

No agregar:

- Login
- Registro
- Navbar
- Footer
- Base de datos
- Backend
- Librerías nuevas

Usar únicamente:

- Next.js
- TypeScript
- Tailwind CSS

---

## Criterios de aceptación

- Responsive
- Código limpio
- Componentes reutilizables
- Fácil de entender
- Sin código innecesario
